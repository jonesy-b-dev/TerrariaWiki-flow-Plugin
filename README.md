# TerrariaWiki-flow-Plugin
A [Flow launcher](https://github.com/Flow-Launcher/Flow.Launcher) plugin to search the [Terraria Wiki.gg](https://terraria.wiki.gg/wiki/Terraria_Wiki) with.
